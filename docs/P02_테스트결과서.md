## 2. 테스트 수행 결과

### 2.1 Memory Manager 유닛 테스트
- **테스트 파일**: `tests/test_memory.py`
- **검증 내용**:
  - 세션 간 대화 이력 독립성 (Pass)
  - 동일 세션 내 대화 누적 및 유지 (Pass)
  - 세션 데이터 삭제 기능 (Pass)
- **결과**: 모든 항목 통과 (3/3)

### 2.2 LLM & 권한 연동 검증
- **Admin 권한**: 토큰의 `groups` 클레임이 `Admin`일 때 관리자용 시스템 프롬프트 주입 확인.
- **User 권한**: 일반 사용자의 경우 친절한 어시스턴트 모드로 동작 확인.
- **Embedding**: `text-embedding-3-small` 호출 시 `dimensions=1024` 파라미터가 올바르게 전달되도록 인터페이스 설계 완료.

## 3. 통합 테스트 시나리오 (실행 가이드)
사용자께서 직접 확인하시려면 아래 절차를 따르세요:
1. 서버 기동 (`PYTHONPATH=. uv run python apps/api/main.py`)
2. 브라우저 로그인 후 얻은 토큰을 사용하여 `/chat` 호출
3. 응답이 한 글자씩 실시간(Streaming)으로 오는지 확인
